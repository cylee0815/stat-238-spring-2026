# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

Graduate course repository for **STAT 238: Bayesian Statistics** at UC Berkeley, Spring 2026. Contains homework assignments with LaTeX problem sets, Python/R computational solutions, and supporting datasets.

## Running Code

**Python scripts** (standalone, run directly):
```bash
python HW1/q4.py
python HW1/q5.py
```

**Quarto notebooks** (.qmd files):
```bash
quarto render HW2/HW2.qmd          # renders to PDF
quarto preview HW2/HW2.qmd         # live preview
```

HW1's `.qmd` uses R with `reticulate` calling Python. HW2's `.qmd` uses a Jupyter Python kernel directly.

**LaTeX documents**:
```bash
pdflatex HW4/HomeworkFour238Spring2026.tex
```

## Key Libraries

- **PyMC** — MCMC/probabilistic programming for posterior sampling
- **ArviZ** — Bayesian diagnostics and visualization
- **NumPy / SciPy** — Numerical integration, distributions (`scipy.stats`), optimization (`scipy.optimize`, `scipy.special`)
- **Matplotlib** — Plotting; output saved as PNG files alongside scripts

## Code Architecture

Each homework folder is self-contained:
- `Homework*.tex/.pdf` — problem statement (do not modify)
- Python scripts or a `.qmd` notebook — solutions
- Data files (`.csv`, `.txt`) — datasets referenced by solutions
- Output PNGs — generated by running the Python scripts

**HW1** uses separate Python scripts per problem (`q4.py`–`q7.py`); plots are saved to PNG files in the same directory.

**HW2+** consolidate solutions into a single `.qmd` notebook rendered to PDF.

## Numerical Conventions

- Computations in log-space to avoid underflow (log-sum-exp trick)
- Fixed random seeds for reproducibility (e.g., `np.random.seed(238238)`)
- Grid-based posterior approximation uses `np.trapz` for normalization before MCMC when feasible
